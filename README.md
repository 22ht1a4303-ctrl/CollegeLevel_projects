# CollegeLevel_projects


E-Commerce Database System
Project Overview

This project is a backend database developed for an E-Commerce application using MySQL. It was created as a SQL capstone project to demonstrate how a real online shopping system manages customers, products, orders, payments, inventory, reviews, and wishlists.

The database is designed to maintain data integrity, reduce redundancy, and support day-to-day business operations through well-structured tables and relationships. It also includes SQL features such as views, stored procedures, triggers, transactions, joins, and subqueries to automate tasks and generate useful business reports.

Project Purpose

The main objective of this project is to understand how a relational database is designed and implemented for a real-world e-commerce application. It demonstrates the use of SQL to manage data efficiently, enforce business rules, automate common operations, and generate reports that support business decisions.

This project also helped improve my understanding of database design, query writing, relationships between tables, data validation, and backend database development using MySQL.
